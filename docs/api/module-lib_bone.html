    (inner) camelCase(str)

    Convert strings connected with hyphen or underscore into camel case. e.g.

    Parameters:

    str string
    Example:
    camelCase('FooBar')   // => 'fooBar'
camelCase('foo-bar')  // => 'fooBar'
camelCase('foo_bar')  // => 'fooBar'
